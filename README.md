# Plink

A mobile app for creating groups and sharing memories with friends. Plink lets you create Parties (groups), invite friends, and share memories together. Members can add posts and upload media to any shared Link.

## Features

- User authentication with profile setup
- Create and join Parties with avatars and banners
- Share Links within Parties
- Add posts and upload images to Links
- Manage Party and Link members

## Tech Stack

- **Expo / React Native** - Mobile framework
- **TypeScript** - Type safety
- **NativeWind** - Tailwind CSS for React Native
- **Supabase** - Database, auth, and file storage

## Setup

```bash
npm install
npm run start
```

Requires a .env file with your Supabase credentials. Run a local build of the app using an emulator or Expo Go app on a mobile device.
